# ⌨️ Keystroke Overlay

> Um overlay moderno para exibir em tempo real as teclas e botões do mouse pressionados durante jogos, gravações de vídeos, lives e apresentações.

---

## 📖 Sobre o Projeto

O **Keystroke Overlay** é um projeto desenvolvido totalmente em **Python** utilizando **Tkinter** e APIs nativas do Windows para criar uma janela transparente e sempre visível na tela.

Seu objetivo é mostrar, em tempo real, quais teclas estão sendo pressionadas pelo usuário, permitindo que espectadores acompanhem exatamente todos os comandos executados durante uma gameplay.

O programa foi desenvolvido pensando em desempenho, simplicidade e compatibilidade com jogos.

Durante sua criação foram implementados diversos layouts diferentes para atender vários tipos de jogadores.

# 📋 Layouts Disponíveis

| Layout | Descrição |
|---------|-----------|
| WASD + Mouse | Ideal para FPS e jogos competitivos |
| Teclado Completo | Exibe praticamente todo o teclado |
| Numpad | Apenas teclado numérico |
| Parte Direita | Área direita do teclado |
| Mão Esquerda | Teclas utilizadas pela mão esquerda |
| Mão Direita | Teclas utilizadas pela mão direita com cores por dedo |

---

# 🎨 Sistema de Cores

O layout da mão direita utiliza um sistema de identificação por dedos.

| Cor | Dedo |
|------|------|
| 🔴 Vermelho | Mindinho |
| 🟢 Verde | Indicador |
| 🟡 Amarelo | Médio |
| 🔵 Azul | Polegar |
| 🟦 Turquesa | Anelar |

Quando uma tecla é pressionada, ela muda automaticamente para vermelho indicando sua ativação.

# 🚀 Como Funciona

Após iniciar o programa:

1. O menu principal é aberto.
2. Escolha o layout desejado.
3. O overlay aparece na tela.
4. Posicione onde preferir.
5. Abra qualquer jogo.
6. Todas as teclas serão exibidas em tempo real.
